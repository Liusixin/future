## 项目介绍
利用手头有一套前端页面,然后在百度上找了一个免费网站设计了一下logo.这样,一个曾经理想做一名前端程序员如今对前端已经一窍不通的家伙就开始了他的代码书写,网站的slogan是--`逼乎,和世界分享你刚编的故事`.
为了简化spring数不胜数的配置文件,在这个项目的第一版中使用了springboot进行开发.在这个项目中尽量不写任何配置文件预计在8.23日之前写完,写不完我就是我猪.

## 开发工具
- git, mysql, IDEA,jdk8
